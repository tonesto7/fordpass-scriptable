Installation

Install the widget tool in just three steps.

1. Install Scriptable

Download Scriptable ↗

If you have not downloaded Scriptable from the AppStore yet, please click on the download button.

2. Copy Installer Script

Copy Installer

Copy the widget tool script to the clipboard by clicking this button.

3. Install Ford Widget Tool

Open Scriptable

Open a new Scriptable script, paste the code and press run (▶).
